# My Diary

A beautiful, modern web-based diary application where users can create, view, and manage their personal diary entries.

## Features

- **User Authentication**: Login and registration system
- **Create Diary Entries**: Add new diary posts with titles and descriptions
- **View Diary Feed**: Browse all diary entries in a clean, card-based layout
- **Detailed View**: Click on entries to view them in full detail
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Modern UI**: Glass-morphism design with beautiful background images
- **Fixed Navigation**: Easy access to post creation and logout from any page

## Technologies Used

- **Frontend**: HTML5, CSS3, Bootstrap 5
- **Styling**: Custom CSS with glass-morphism effects
- **Fonts**: Google Fonts (Comfortaa)
- **Icons**: Bootstrap Icons
- **Version Control**: Git & GitHub

## Pages

1. **Login Page** (`index.html`): User authentication
2. **Registration Page** (`Registration.html`): New user signup
3. **Feed Page** (`feedpage.html`): View all diary entries
4. **Post Creation** (`post.html`): Create new diary entries
5. **Entry Detail** (`Dedicatedpage.html`): View individual entries

## Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/akshiththota99/MY-DIARY.git
   cd MY-DIARY
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - No server setup required for the frontend

## Usage

1. **Register**: Create a new account on the registration page
2. **Login**: Use your credentials to access the diary
3. **View Entries**: Browse your diary entries on the feed page
4. **Create Entry**: Click "Post My Day" to add new diary entries
5. **Read Details**: Click "View Post" on any entry to read the full content
